Запуск программы.
1. Собрать проект.
2. Скопировать BD.db в папку с собранным проектом.
3. Запустить SignalForm.exe
   
Запуск программы.
1. Указать параметры сигнала
2. Нажать сгенерировать.
Очистка графика
1. Нажать кнопку очистить.
   
Постраничный вывод сохраненных сигналов кнопки Назад и Вперед.
